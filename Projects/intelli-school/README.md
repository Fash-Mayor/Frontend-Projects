# Assignment 1: School Website Design

## Overview
A fully responsive multi-page website for a school with clean, professional design built with pure HTML and CSS.

## Features

### Pages
1. **Home Page (index.html)**
   - Eye-catching hero section with call-to-action
   - Feature highlights section (Expert Faculty, Modern Facilities, Student Success)
   - Call-to-action section
   - Responsive footer with contact information

2. **About Page (about.html)**
   - School story and history
   - Mission and vision statements
   - Core values (6 values displayed in a grid)
   - Statistics/achievements
   - Programs offered
   - Call-to-action section

3. **Contact Page (contact.html)**
   - Functional contact form with validation
   - Contact information section with multiple contact methods
   - FAQ section with common questions
   - Office hours information

### Design Highlights

**Color Scheme:**
- Primary Color: Deep Blue (#2c3e50)
- Secondary Color: Bright Blue (#3498db)
- Accent Color: Gold (#f39c12)
- Background: Light Gray (#f8f9fa)

**Typography:**
- Headings: Poppins (Bold, modern)
- Body Text: Lato (Clean, readable)

**Layout:**
- Flexbox-based responsive design
- Mobile-first approach
- CSS Grid for multi-column layouts
- Smooth hover effects and transitions

### Features
- Sticky navigation bar with responsive hamburger menu
- Gradient backgrounds on hero and CTA sections
- Smooth animations and transitions
- Mobile-responsive design (tested on all screen sizes)
- Form validation on contact page
- Professional typography and spacing

## Technical Stack
- **HTML5:** Semantic structure
- **CSS3:** Modern styling with Flexbox and Grid
- **JavaScript:** Mobile menu and form handling
- **Fonts:** Google Fonts (Poppins and Lato)

## How to Use
1. Open any HTML file in a web browser
2. Navigate between pages using the navigation bar
3. Fill out the contact form on the contact page
4. Mobile menu appears on screens smaller than 768px

## Responsive Breakpoints
- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: Below 768px
- Extra Small: Below 480px

## Design Choices
1. **Professional Color Palette:** The blue and gold combination conveys trust, education, and excellence
2. **Clean Typography:** Two-font system (one serif for headings, one sans-serif for body) creates hierarchy
3. **Generous Spacing:** Ample whitespace ensures readability and professional appearance
4. **Interactive Elements:** Hover effects provide visual feedback without being overwhelming
5. **Responsive Grid:** Flexible layouts adapt smoothly to all screen sizes
